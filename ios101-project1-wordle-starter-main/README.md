#Project 1 - Wordle Pt. 1 Starter
This is the starter project for iOS 101 Project 1 - Wordle Pt. 1

Submitted by: Zachary Nalapo

Wordle is a puzzle app where a random word is pulled from a pool of words and the user has a limited number of tries to guess the word. 

Time spent: 2 hours

##required features

- [x] Displaying the keyboard on the wordle app
- [x] Allowing a user to input the letters on the keyboard or delete the last character with the delete key
- [x] A random word is generated from a pool of words and the correct letters in the word are highlighted yellow if in the word, but not in the right place, or green if the letter is in the right place.

##Optional features

- [ ] Add image asset to the icon for the home screen
- [ ] Run the app on a personal device
